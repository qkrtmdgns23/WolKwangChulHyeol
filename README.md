# Folder Structure
|폴더 이름        |폴더 내용물                   |폴더 구조                        | 
|----------------|-----------------------------|--------------------------------|
|Scripts         |스크립트 (코드)               |namespace, feature를 구분해서 분류|
|Game            |프리팹, 스크립트가 포함된 오브젝트들|Entity 기준으로 분류|
|Scenes          |씬, 터레인, 네브매쉬 등|Scene 기준으로 분류|
|UI              |UI 프리팹|스크린 기준으로 분류|
|Asset Packs     |불러온 모든 리소스들|파일 그대로|
|Audio           |오디오 파일들||
|Models          |FBX, Mesh, Animation, 등||
